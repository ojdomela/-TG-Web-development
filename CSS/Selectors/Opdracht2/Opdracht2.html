Opdracht 2:

CSS pas je toe met een bepaalde syntax. Hoe ziet zo'n syntax eruit? Kun je ook uitleggen wat elk element uit de syntax betekent?

CSS syntax bestaat uit een Selector en een Declaration.

Selector: hiermee selecteer je welk element je aanpast. Je kan elementen selecteren op meerdere manieren, bijv. per class of per element.

Declaration: wat je met het geselecteerde element doet. Deze bestaat uit een property en een value die je die property aanwijst; bijv. de property achtergrond-kleur de value grijs geven.

In de CSS van deze pagina is er een selector p met property: color die dan de value:red is gegeven.
