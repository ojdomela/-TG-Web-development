Welke position properties zijn er? Leg voor elke position property uit wat het doet. Geef ook aan welke default position waarde elementen hebben die je gebruikt in html.

  • Static - de defaultwaarde voor HTML elementen. top/right/bottom/left/z-index doen allemaal niks als het element deze property heeft
  • Relative - top/right/bottom/left verzetten het element t.o.v. waar het normaal in het document zou staan
  • Absolute - het element wordt uit de normale HTML-volgorde gehaald en vervolgens met top/right/bottom/left op een vaste plaats gezet binnen de pagina
  • Fixed - het element wordt uit de normale HTML-volgorde gehaald en vervolgens met top/right/bottom/left op een vaste plaats gezet binnen de viewport. Deze blijft op dezelfde positie op het scherm, ook als de pagina omhoog/naar beneden scrollt.
  • Sticky - hybride tussen relative/fixed - positioneert initieel net als position:relative maar blijft binnen de viewport als je er voorbij scrollt
