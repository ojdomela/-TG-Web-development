Leg uit wat de volgende termen betekenen: content, padding, margin en border.

  • Content: De 'inhoud' van de CSS box. Hierin zitten de HTML elementen die je samen een bepaalde plaats op je website wil geven.
  • Padding: De ruimte tussen de content en de rand van de CSS box.
  • Border: De rand van de CSS box.
  • Margin: De ruimte tussen de rand van de CSS box en de andere elementen op je pagina.
