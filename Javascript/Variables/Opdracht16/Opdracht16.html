sentence = This string is a sentence to test different string methods.

2. concat() - sentence.concat(' This is an extra test string after the first test string.') - zet de gegeven string achter de huidige zin en selecteert het resultaat.
3. slice() - sentence.slice(0, 33) = This string is a sentence to test - Pakt een stuk van de string met begin en eind-index als parameter.
4. split() - Zet de string in een array, met alles tussen de gegeven items in als 1 item in het array. - sentence.split(' ') = Array of single words. sentence.split('') = Array of single letters.
5. substr() - sentence.substr(29, 4) = test - Werkt net als slice() om een stuk van de string te selecteren, maar dan met een begin-index en een lengte ipv begin + eind-index als parameters.
